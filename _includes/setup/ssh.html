Accessing SSH

University of Florida has a supercomputer cluster with latest generation of processors for memory intensive computings called High-Performance Cluster (or HiperGator in short). We will connect to HiperGator from your computer using SSH. The SSH protocol (also referred to as Secure Shell) is a method for secure remote login from one computer to another.

  • Windows
  • MacOS
  • Linux

Windows does not have SSH client by default. However, SSH can be added from Optional Features in recent version of Windows 10. This can be performed through GUI or commandline.

Using graphical interface

  1. Search for optional in windows search (Win + S) and click on Manage Optional features.
  2. Check if OpenSSH Client is present in the list of installed features. If it is present, SSH client is installed, so you can skip rest of the steps.
  3. Click on Add a feature. Search for ssh. Click checkbox next to OpenSSH Client and click Install.
  4. Wait for the install to complete.

Using command line interface

  1. Open Powershell in administrative mode.
    • Method 1: Search for powershell in windows search (Win+S). Right-click on Windows powershell and click Run as administrator.
    • Method 2: Right click on start button, then click Windows Powershell (Admin).
    • Method 3: Press keyboard shortcut Win+X, then click Windows Powershell (Admin).
  2. Click Yes button in UAC prompt.
  3. Enter the following: 
    4. Get-WindowsCapability -Online | ? Name -like 'OpenSSH*'
  4. If the value of state under OpenSSH.Client is Installed, SSH client is present and you can skip next steps.
  5. Enter the following: 
    6. Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0
  6. If the output includes Online: True, then SSH Client is successfuly installed.

To start using SSH, Open Powershell. For this, search for powershell in windows search (Win+S) and click on Windows powershell. Alternatively, right-click start button or press Win+X and click Windows Powershell.

For ease of access, you may want to pin Powershell in your taskbar for this workshop.

Note: Earlier verions of Windows 10 or previous editions of Windows do not have Optional Features. For such Windows devices, we recommend downloading Putty from putty.org. Please contact the instructors prior to the workshop if you are having difficulty to access SSH Client.

SSH client is avilable by default in all versions of macOS, so there is no need to install anything.

To start using SSH, open the Terminal either from /Applications/Utilities or by searching for terminal in Spotlight (Cmd+Space).

For ease of access, you may want to pin Terminal in your dock for this workshop.

SSH client is avilable by default in most linux distros, so there is no need to install anything.

To start using SSH, open the Terminal from Application list. Ubuntu based distros can open Terminal with keyboard shortcut Alt+Ctrl+T.

For ease of access, you may want to pin Terminal in desktop panel for this workshop.
