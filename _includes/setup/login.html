Remote login by SSH

Note: If you are using PUTTY to connect, use this guide instead.

  1. Open Terminal or Powershell.
  2. Type the following command and press Enter key. Do not foret to replace <username> with your provided username. 
    ssh <username>@hpg.rc.ufl.edu
  3. If you are connecting for the first time, you will be prompted to verify the ECDSA keys as shown below. 
    The authenticity of host 'hpg.rc.ufl.edu (128.227.221.250)' can't be established.
ECDSA key fingerprint is SHA256:z5zVt98Qo+rj0wiWrUhsvuTxVAjFBBePQwcRLq5g2ZI.
Are you sure you want to continue connecting (yes/no/[fingerprint])?
  4. Type yes and press Enter key.
  5. Next, you will be prompted to enter password as shown below. Type the provided password and hit Enter key. 
    <username>@hpg.rc.ufl.edu's password:
    Note that, as you type in the password, the typed letters to not appear on the screen. That is by design.

If you typed your password correctly, you will be able to access HiperGator. The following welcome screen will appear. 

      Welcome to UF Research Computing
      ________________________________

The user agrees to comply with Research Computing's policies.

https://www.rc.ufl.edu/services/procedures/
