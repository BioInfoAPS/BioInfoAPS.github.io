Setup

Setup

To participate in a this workshop, you will need access to the software described below.

We maintain a list of common issues that occur during installation as a reference for instructors that may be useful on the FAQ page.

Install the videoconferencing client

If you haven't used Zoom before, go to the official website to download and install the Zoom client for your computer.

Set up your workspace

Like other bioinformatics workshops, you will be learning by "coding along" with the Instructors. To do this, you will need to have three windows open at the same time, one for the command line shell (powershell/terminal), one for the website (to copy commands), and one window for the Zoom video conference client. In order to see all three at once, we recommend using one of the following set up options:

  • Two monitors: If you have two monitors, plan to have your terminal and website up on one monitor and the video conferencing software on the other.
  • Two devices: If you don't have two monitors, do you have another device (tablet, smartphone) with a medium to large sized screen? If so, try using the smaller device as your video conference connection and your larger device (laptop or desktop) to follow along with the terminal and the website.
  • Divide your screen: If you only have one device and one screen, practice having three windows (the zoom window, a terminal/powershell and the website) open together. How can you best fit both on your screen? Try it out in advance to decide what will work best for you.

Accessing SSH

University of Florida has a supercomputer cluster with latest generation of processors for memory intensive computings called High-Performance Cluster (or HiperGator in short). We will connect to HiperGator from your computer using SSH. The SSH protocol (also referred to as Secure Shell) is a method for secure remote login from one computer to another.

  • Windows 11
  • Windows 10
  • MacOS
  • Linux

Windows 11 does not have SSH client by default. However, SSH can be added from Optional Features. This can be performed through GUI or command line.

Using graphical interface

  1. Search for optional in windows search (Win + S) and click on Manage Optional features.
  2. Check if OpenSSH Client is present in the list of installed features. If it is present, SSH client is installed, so you can skip rest of the steps.
  3. Click on Add a feature. Search for ssh. Click checkbox next to OpenSSH Client and click Install.
  4. Wait for the install to complete.

Using command line interface

  1. Open Windows Terminal in administrative mode.
    • Method 1: Search for terminal in windows search (Win+S). Right-click on Windows Terminal and click Run as administrator.
    • Method 2: Right click on start button, then click Windows Terminal (Admin).
    • Method 3: Press keyboard shortcut Win+X, then click Windows Terminal (Admin).
  2. Click Yes button in UAC prompt.
  3. Enter the following: 
    4. Get-WindowsCapability -Online | ? Name -like 'OpenSSH*'
  4. If the value of state under OpenSSH.Client is Installed, SSH client is present and you can skip next steps.
  5. Enter the following: 
    6. Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0
  6. If the output includes Online: True, then SSH Client is successfuly installed.

    7. You can close Windows Terminal now.

To start using SSH, Open Windows Terminal. For this, search for terminal in windows search (Win+S) and click on Windows Terminal. Alternatively, right-click start button or press Win+X and click Windows Terminal.

For ease of access, you may want to pin Windows Terminal in your taskbar for this workshop.

Transferring files

Transfering files to and from remote server and your local computer requires a separate transfer protocol, such as Samba or SFTP. The complete list of data transfer methods available for HiperGator is listed in its wiki page. We recommend use of SFTP (Secure File Transfer Protocol) for accessing files for this workshop. Although commandline version of SFTP are often bundled with SSH, we recommend use of graphical softwares for ease of use if you are not familiar with sftp commands.

  • Windows
  • MacOS
  • Linux

You can use Filezilla and WinSCP for uploading or downloading data from remote SFTP server.

You can use Filezilla and Cyberduck for uploading or downloading data from remote SFTP server.

You can mount SFTP as virtual filesystem using sshfs and access the files using file any file manager. 

sudo apt install sshfs
mkdir /<mountpoint>
sshfs <username>@hpg.rc.ufl.edu:/blue/general_workshop/<username> /opt/<mountpoint>

Alternatively, you can use Filezilla for uploading or downloading data from remote SFTP server.


The following instructions are for Filezilla.

  1. Download Filezilla using this link.
  2. Install Filezilla and then run the program.
  3. Fill the following information in quickconnect panel and press Quickconnect button.
    • Host: hpg.rc.ufl.edu
    • Username: provided username
    • Password: provided passsowd
    • Port: 22
  4. Click OK if prompted to certify the RSA key fingerprint.
  5. Change remote site to the following. Do not forget to replace <username> with the provided username. 
    /blue/general_workshop/<username>

We will generate a tree in Newick format. You can download a tree viewing software such as Figtree to visualize the tree. Figtree is available for download for Windows, MacOS and Linux from its github repository.
